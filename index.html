Module 2

Setting Up Dropbox

This foundational module guides you through the entire process of getting started with Dropbox. You will learn to install the application on your devices, configure your account for optimal use, and perform your first file synchronization to see the magic of cloud storage firsthand.

Cross-Platform Installation

Install Dropbox on any OS, including Windows, macOS, and Linux, for a consistent experience.

Initial Account Setup

Secure your account and customize preferences for notifications and bandwidth usage.

Understanding Sync

Grasp core concepts of cloud sync and what status icons for files mean.

Folder Management

Master the basics of creating and organizing folders for an efficient file system.

What you will learn:

  • Install and configure the Dropbox desktop and mobile applications.
  • Successfully sync your first files and folders to the cloud.
  • Understand the difference between local files and online-only files.
  • Set up a basic folder structure for personal or project files.
